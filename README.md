# Nova Color Picker

A premium Chrome Extension for designers and developers to pick any color from their screen and copy it in multiple formats.

## Features

- **EyeDropper Integration**: Select colors from any webpage or even outside the browser (on supported systems).
- **Multiple Formats**: Instantly get HEX, RGB, and HSL values.
- **Color History**: Automatically saves the last 20 colors picked.
- **One-Click Copy**: Copy values directly to your clipboard.
- **Glassmorphism Design**: Sleek, modern, and dark-themed UI.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`.
2. Enable **Developer mode** in the top right corner.
3. Click **Load unpacked**.
4. Select the `Nova Color Picker` folder.

## Technologies Used

- JavaScript (ES6+)
- CSS3 (Custom Properties, Glassmorphism, Animations)
- Chrome Extension API (Manifest V3, Storage API)
- EyeDropper API
